Kirei is the Japanese translation of "Beautiful", so the name of this site can also be Beautiful Valorant, this site contains latest information about the game such as agents that can be played, maps, weapon that can be used, cards to more identify yourself and bundles to elevate the gaming experience.

This site, which was created with Angular and Bootstrap 5, has simple navigation, the recognizable color palette from the brand itself, and gorgeous in-game assets taken directly from the open-source Valorant API, which can be found here.

Objectives & Goals

With this project, I'm mostly trying to gain experience calling open APIs. Since I play the game myself and thought it would be wonderful to combine two things I enjoy doing, I chose the public Valorant API.

Problems Encountered & The Path to Solving them

With some API endpoints, it was simple for me to iterate through the data and display it on the HTML page, but with others, it was more difficult. I finally found a solution by using Angular's KeyValue pipe, which was simple to learn and effectively fixed my problem of trying to access some of the data that I received from the API.
