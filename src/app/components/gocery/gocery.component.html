Go-cery is a web e-commerce application dedicated to selling groceries, it allows users to see different products sorted by their category, add them to a cart, and place orders.

Go-cery (2021) was a project I built with Bernie Inociete and Euriel John Mora for a class project, back then, I wanted to use as much Vanilla CSS as possible as I wasn't too confident back then with using Bootstrap or Materialize (frameworks that were widely used by our classmates).

Objectives & Goals

Go-cery (2021) needed a thorough redesign because I believe the previous project lacked several UI fundamentals that I was unaware of at the time. My first experience with responsive web design was also with Gocery (2021), and with this new project, I wanted to show how my knowledge about responsiveness had changed after a year.

Additionally, I wanted to utilize a framework this time rather than just plain CSS. I originally wanted to use Angular Material, but I ultimately settled on Bootstrap 5 as I believe this will serve me the best.

Problems Encountered & The Path to Solving them

In contrast to the previous project, I wanted the categories and the stores to display their products on a single HTML file. I used the LocalStorage to accomplish this goal; the products component would look up an item's value in the LocalStorage and then display products either when a user clicked on a category or a store.

Lessons Learned âœ¨

I learned in here that Bootstrap has a lot of utility classes that make our lives as a developer easier, previously I had to come up with a class name whenever I want a div container to be flexed but with Bootstrap, it is easy as typing 3 classes to an existing div to emulate a flex display.

Although I did not use it that much in this project, Bootstrap's grid system is also very powerful, it's easy to understand and helps us even more to build layouts that are truly responsive.

I now see what gives Bootstrap its fame, you really can build User Interfaces much faster. I now appreciate it more and I can see myself using it again for my future projects.
