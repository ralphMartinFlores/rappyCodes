This is a slightly modified version of Frontend Mentor's The Planets challenge, this simple site contains beautiful images and information about the planets in our Solar System.

Because I decided that I could have the similar experience with tabs, I developed the project using Angular and Angular Material for easy Component creation for each Planet and easy routing. This slight alteration alone sets it apart from the version of Frontend Mentor.

Objectives & Goals

I mainly wanted to be able to transfer an Angular-made static site to Netlify and just recreate the challenge using the framework.

Problems Encountered & The Path to Solving them

Although I didn't have many issues while developing this project, I did run across one as I was about to launch the site. Once I developed the project and uploaded it to Netlify, the routing stopped working; it turns out that it is more difficult than not using a framework.

Fortunately, I was able to resolve that one issue after discovering an online guide that addresses how to correctly publish an Angular project to Netlify.
